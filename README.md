<h1> Teste automatizado do site tricentis </h1>
Automação utilizando o Selenium Webdriver, Cucumber, TestNG, Log4j, Page Objects e PageFactory.

Obs.: Caso exista algum firewall ou proxy bloqueando a utilização do Webdrivermanager, será necessário descomentar a linha 14 e comentar a linha 15 da classe DriverManager que está no pacote "utils".
